# HyperGrok-UI

Improves Grok’s prompt input UI by expanding the input bar to full width and increasing textarea height before scrolling. This makes long prompts significantly easier to write, edit, and refine in Grok Imagine.

---

# What This Extension Does

### Default Grok prompt box limitations
- Input bar does not use the full available width
- Long prompts feel cramped horizontally
- Prompt box becomes scrollable after only ~6 lines
- Editing long prompts requires constant scrolling

### HyperGrok-UI fixes these issues by
- Expanding the prompt input bar to full width
- Increasing prompt box expansion height
- Allowing approximately **15–16 visible lines** before scrolling
- Making long prompt workflows much cleaner and easier

### Best for
- Long image prompts
- Prompt engineering
- Frequent prompt tweaking
- Grok Imagine power users

# Installation

## Step 1 — Download the Extension

Click the green **Code** button on GitHub and select:

**Download ZIP**

Extract the ZIP somewhere on your computer.

Example:
```text
Desktop/HyperGrok-UI
````

---

# Chrome Installation

1. Open Chrome
2. Go to:

```text
chrome://extensions
```

3. Enable **Developer Mode**

4. Click **Load unpacked**

5. Select the extracted extension folder

Example:

```text
Desktop/HyperGrok-UI
```

6. Done

Open Grok and refresh the page.

# How to Verify It Is Working

Open Grok Imagine.

You should notice:

* Wider prompt input bar
* More horizontal space
* Taller expanding prompt area
* Less scrolling for long prompts

If nothing changes:

* Refresh Grok
* Make sure extension is enabled
* Disable conflicting UI extensions

---

# Known Limitations

This extension relies on Grok’s current webpage class names.

If Grok updates its UI structure or CSS classes, HyperGrok-UI may stop working until updated.

This is normal for CSS-based UI tweaks.

---

# Updating the Extension

If a new version is released:

1. Download latest version

3. Go to browser extensions page
4. Click **Reload** on the extension
